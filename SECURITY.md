# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any EZScale SDK, please report it responsibly.

**Email:** security@ezscale.cloud

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and provide a detailed response within 5 business days.

## Supported Versions

| SDK    | Version | Supported |
|--------|---------|-----------|
| PHP    | 1.x     | Yes       |
